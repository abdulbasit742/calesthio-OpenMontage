# Archive Register Checks

This file gives a short check list for the archive register reference.

## Use After

- storage receipt report is ready
- register note is added
- register guide is added

## Checks

- receipt status is `received`
- receipt ID is present
- register ID is present
- package label is present
- owner is present
- reviewer is present
- storage owner is present
- storage location is present
- package index includes the register ID
- register note is kept with the package

## Register ID Example

```text
ARCHIVE-STORAGE-REGISTER-001
```

## Review Note

The register reference is ready when the package can be found again by register ID and the receipt ID is linked beside it.
